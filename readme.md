# Uppgift 2 - Typescript
Det här projektet är en enkel att göra-applikation byggd med TypeScript. Syftet är att användaren ska kunna skapa, hantera och spara sina uppgifter på ett smidigt sätt direkt i webbläsaren.

**Funktioner**
- Lägg till nya uppgifter med text och prioritet
- Markera uppgifter som klara
- Kunna ta bort slutfärda uppgifter
- Visa en lista med alla uppgifter
- Spara och ladda uppgifter automatiskt med LocalStorage
- Responsiv design som fungerar på både mobil och dator
